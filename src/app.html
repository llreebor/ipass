Secure Mobile Credentials

  • Work with any Bluetooth smartphone

  • Challenge-response authentication

  • Intuitive apps for iOS and Android

  • Simple enrollment to all access systems

  • Optional multi-factor authentication

  • iPass mobile credentials allow any Bluetooth smartphone to be used for physical access via a simple and intuitive app for iOS and Android.

  • iPass mobile credentials optionally support multi-factor authentication, by checking a PIN code or performing facial recognition or fingerprint matching.

  • iPass utilities enroll credentials into any access system, enabling secure access via users’ phones without the need for smart cards or access keys.

iPass Mobile Credentials

Leverage Bluetooth smartphones or watches as secure access credentials with iPass Mobile Credentials. Our intuitive iPass App, available for both iOS and Android, allows for quick and easy enrollment and credential generation. With support for multi-factor authentication, including PIN codes, facial recognition, and fingerprint matching, our mobile credentials offers flexible security levels to meet the most stringent security requirements

The iPass App

The iPass Access app for iOS and Android allows any Bluetooth enabled your Phone to be used as an access credential for iPass multi-technology access readers.
