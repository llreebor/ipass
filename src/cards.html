Cryptographic
Access Cards

  • Unbreakable cryptographic credentials

  • State-of-the-art public key cryptography

  • Compatible with all access systems

  • Simplfied "No PKI" architecture

  • Easy enrollment into popular PACS

  • iPass access cards use state-of-the-art public key cryptography to implement unbreakable credentials without complex public key infrastructure.

  • iPass access cards couple unique private keys that never leave the card with matching public keys that can be exposed without risk.

  • iPass access cards and iPass readers are compatible with all modern access control systems using Wiegand or OSDP connectivity.

Our Cards

Contactless Card

iPass PKC10 contactless access cards work with iPass readers to perform challenge-response authentication using a state-of-the-art Ellipic curve algorithm.

  • Contactless smart card

  • Public key cryptography

  • Contactless NFC interface

  • Challenge-response authentication

  • Printable blank white finish

  • Contactless protocol

    ISO 14443A

  • Communication frequency

    13.5MHz

  • Sensing distance

    2.5~10mm

  • Public-key algorithm

    ECC P-256

  • Dimensions (inches)

    3.38x2.13x0.03

  • Dimensions (mm)

    85.5x54x0.8

  • Card material

    PVC laminate
